# Design System Guidelines

## Principles

- Prefer Stack for vertical spacing and layout
- Use primary buttons only for the main call to action
- Use secondary buttons for supporting actions
- Cards should group related information

## Button

Variants:
- primary
- secondary
- ghost

Avoid:
- multiple primary buttons in the same section

## Card

Cards contain:
- title
- description
- actions

## Stack

Use Stack instead of custom flex column layouts.
